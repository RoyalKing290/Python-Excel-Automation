# Python-Excel-Automation
Automate Excel files, clean data, and generate reports using Python

This Python script automates Excel files:
- Cleans messy data
- Merges multiple files
- Generates ready-to-use reports
- Provides summary totals & by-product aggregation

## How to Use
1. Place your CSV file in the folder
2. Run `excel_automation.py`
3. `report.xlsx` will be generated automatically

## Features
- Reusable Python script
- Handles multiple files
- Cleans errors and inconsistencies
- Professional reports for decision-making
